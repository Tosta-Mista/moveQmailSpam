# moveQmailSpam
## Description
Simple script used to move mail tagged as spam to .Spam folder.

## Usage
Run script
```bash
$ ./moveQmailspam.sh -d <domain.tld>
```

Get version
```bash
$ ./moveQmailSpam.sh -v
Version 0.0.2
```

Get help
```bash
$ ./moveQmailSpam.sh -h
Usage: command -dhv args
```
